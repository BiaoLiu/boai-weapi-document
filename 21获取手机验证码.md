#2.1.获取手机验证码

###### URL

> auth/get_authcode/

###### HTTP请求方式

> POST

###### 是否登录

> 否

###### 请求参数

>|参数|必选|类型|默认值|说明|
|:-----: |:-------|:-----:|:-----:|:-----:|
|mobile|true|string| |手机号码 |
|device_id|true|string| |设备id|
|ver|true|string| |版本号|
|ts|true|string| |时间戳 |




###### 返回字段

>|返回字段|字段类型|说明 |
|:------:|:------:|:------------:|
|is_success|bool |是否发送成功 |


###### 接口示例

```
{
 "data":
      { 
        "is_success": true,
      },
 "retcode": "10000",
 "msg": ""
}
```
